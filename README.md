# NEXUS

**Library Management System — C.W.W. Kannangara Central College, Mathugama**

> _Empowering Readers, Digitizing Dreams._

---

A mobile-first, student-built library platform replacing a 2012-era spreadsheet system. Serving 4,198 students, 188 teachers, and 77 staff with automated workflows, intelligent analytics, and a digital resource vault.
